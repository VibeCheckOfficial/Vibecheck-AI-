# Privacy Policy

**vibecheck VS Code Extension**

Last updated: January 5, 2026

## Summary

- ✅ Core analysis runs **100% locally** on your machine
- ✅ **No telemetry** is collected without opt-in
- ✅ **No code** is sent to external servers without explicit action
- ✅ Premium features require **explicit API key configuration**

## Data Collection

### What We DON'T Collect

- Your source code
- File contents
- Project structure
- Personal information
- Usage patterns
- Error reports (unless you opt-in)

### What We MAY Collect (with API key)

When you configure a vibecheck API key and use premium features:

- Anonymized scan metadata (score, issue counts)
- Feature usage statistics
- Error reports for debugging

This data is used solely to:

- Provide premium features
- Improve the product
- Debug issues

## Local Analysis

The following features run entirely on your machine:

- Workspace scanning
- Inline diagnostics
- CodeLens warnings
- AI code validation (using your own OpenAI key)
- Score calculation

No network requests are made for these features.

## Premium Features

When you use premium features with an API key:

- Reality Mode results may be stored temporarily for report generation
- Compliance reports are processed on our servers
- Team dashboard data is synced to your organization's account

## Third-Party Services

The extension may optionally use:

- **OpenAI API** (if you provide your own key) for AI intent verification
- **vibecheck API** (if you provide an API key) for premium features

## Data Security

- All API communication uses HTTPS
- API keys are stored securely in VS Code's secret storage
- We do not share data with third parties

## Your Rights

You can:

- Use the extension without any API keys (fully offline)
- Delete your API key at any time
- Request deletion of any data associated with your account

For privacy questions: privacy@vibecheckai.dev

## Changes

We may update this policy. Changes will be noted in the CHANGELOG.
